# Cerca d'artista (ex. Radiohead)
curl -G "https://musicbrainz.org/ws/2/artist" --data-urlencode "query=Radiohead" -H "User-Agent: CordovaMaterialize/1.0 (tu@email.com)" -H "Accept: application/json"

# Cerca de cançó (ex. Creep)
curl -G "https://musicbrainz.org/ws/2/recording" --data-urlencode "query=recording:Creep" -H "User-Agent: CordovaMaterialize/1.0 (tu@email.com)" -H "Accept: application/json"

Nota: quan proves al navegador pot aparèixer CORS; a l'app Cordova normalment funciona sense proxy.
